# 🔥 AI-Powered Fire & Control Decision Support System

🚀 Turning Emergency Calls into Real-Time Actionable Decisions

---

## 📌 Overview

This project is an **AI-powered real-time fire emergency control system** that transforms raw emergency calls into structured intelligence and actionable insights.

It helps emergency responders:
- Understand incidents faster  
- Eliminate duplicate reports  
- Prioritize emergencies intelligently  
- Take informed actions in real-time

## ❗ Problem Statement

Current emergency fire response systems rely heavily on **manual call handling**, which leads to:

- ⏱ Delayed response time  
- ❌ Incomplete and inconsistent information  
- 🔁 Duplicate incident reporting  
- 🧠 Lack of intelligent decision support  

---

## 💡 Solution

Our system uses **AI + real-time processing** to:

- Convert audio calls → structured data  
- Extract:
  - 📍 Location  
  - 🔥 Type of fire  
  - 👥 People involved  
  - ⚠️ Hazards  
- 🔗 Merge multiple calls into one incident  
- 🚨 Assign priority (High / Medium / Low)  
- 🤖 Generate AI insights  
- 🎯 Recommend actionable steps

## 🧠 Key Features

- 🧠 AI-based Information Extraction  
- 🔁 Incident Merging System  
- 🚨 Priority Classification

## 🏗️ System Architecture

1. Audio Input (Emergency Call)  
2. Speech-to-Text + Translation  
3. AI Extraction (LLM)  
4. Incident Matching & Merging  
5. Priority & Insights Generation  
6. Real-Time Dashboard Update
